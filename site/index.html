# install (macOS / Linux)

$ curl -fsSL https://forge.yourdomain.com/install.sh | sh

Checks your setup, installs the forge CLI. Windows: pip install blueshark-forge.

what it is

forge drives a coding agent on whatever weights you point it at. The same agent, memory, verification, and tools work across any model — you swap the weights, not the harness. It's plain Python, stdlib-only, zero runtime dependencies.

  • Model-agnostic. Local models via Ollama or llama.cpp, or frontier models (OpenAI, Anthropic) with your own key.
  • Harness-lift. Memory, verification, escalation, and constrained output make a small model genuinely useful — and forge measures the base-vs-harness gain, so it's evidence, not a claim.
  • Local & private. Runs on your machine. Your files never leave it. No account, no server, works offline.
